# BlogAura

Application de blog moderne développée avec React et Tailwind CSS.

## Démarrage rapide

```bash
# Installer les dépendances
npm install

# Lancer en développement
npm run dev
```

Ouvrir http://localhost:5173

## Comptes de test

| Rôle | Email | Mot de passe |
|------|-------|--------------|
| Admin | admin@blog.com | admin123 |
| User | user@blog.com | user123 |

## Fonctionnalités

- Affichage des articles avec aperçu
- Système de commentaires avec pagination
- Réactions emoji avec compteurs
- Dark/Light mode
- Filtrage par tags
- Protection des routes (admin)

## Technologies

- React 19
- Vite 7
- Tailwind CSS 4
- React Router DOM 7
